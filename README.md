# sendMailCmd
## A command line tool Send mail with Golang
```bash
 ./onsiteinspectionfork8s 
  -body string
        mail's body (default "sendmail ok")
  -fm string
        mail's body format example: text/html text/plain (default "text/html")
  -from string
        from user
  -host string
        mail server address
  -pass string
        user pass
  -port string
        mail server port (default "25")
  -subject string
        mail's subject (default "Test")
  -to string
        send to user example: xx@xx.com;yy@yy.com
```
